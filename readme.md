# Ballpoint

A Rust implementation of Inkle's [Ink][ink]

## Installation

Ballpoint supports an MSRV of Rust 1.50.0

To start, add Ballpoint to your project's `Cargo.toml`
```toml
ballpoint = "0.1.0"
```
